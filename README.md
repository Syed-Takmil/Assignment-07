
<h2 align="center"> Project Name:Kin Keeper</h2><br>
<i>Description:</i> 
<br>
<b>
  <i>
    KeenKeeper is a social interaction tracking platform that allows users to manage and monitor their connections. It records different types of interactions such as calls, text messages, and video calls, and presents them in an organized and visual way. Users can explore interaction history through timelines and gain insights using dynamic Pie chart.
  </i>
</b>
Technologies Used: <br>
<ol type="i">
  <li>React.js</li>
  <li>React Router DOM </li>
  <li>Tailwind CSS</li>
  <li>Daisy UI</li>
  <li>React Toastify </li>
  <li>Recharts</li>
</ol>
 ##3 Features of the Project: 

<ol>
<li>
    Interaction Tracking System
Records and categorizes user interactions (calls, texts, video calls) with friends in a structured way.
</li>
<li>
  Dynamic Statistics Dashboard
Displays interaction data in a responsive pie chart that updates automatically based on user activity.
</li>
<li>
  Timeline & Search Functionality
Shows a chronological list of all interactions, with filtering and search options to quickly find specific records.
</li>
</ol>
